白无常AI 图片提示词与生图助手隐私政策

生效日期：2026 年 5 月 6 日

白无常AI 图片提示词与生图助手是一款 Chrome 扩展，界面品牌为 AIxLab。本政策说明扩展如何处理用户数据。

我们处理哪些数据

  • 用户在设置页主动填写的阿里云百炼 DashScope API Key。
  • 用户配置的 Qwen-VL、万相 2.7 模型名称和 API Base URL。
  • 用户主动通过右键菜单选择的网页图片 URL、页面 URL、页面标题和选择时间。
  • 用户选择图片后生成或编辑的中文提示词、English Prompt 和生成图片结果。

数据存储方式

API Key、模型配置和当前选择的图片信息保存在用户浏览器本地的 chrome.storage.local。扩展不使用自建云端服务器保存这些配置。用户可以通过 Chrome 的扩展管理页面清除扩展数据，也可以卸载扩展删除本地数据。

数据如何被使用

扩展仅在用户主动操作时使用数据：当用户右键选择网页图片并点击提取或生成按钮后，扩展会读取该图片，并调用用户配置的 DashScope / 阿里云百炼 API，用于完成图片提示词提取和参考图生图。

数据共享对象

为了提供核心功能，用户选择的图片内容、提示词和必要的 API 请求参数会发送到用户配置的 DashScope / 阿里云百炼 API 端点。扩展不会将用户数据出售给第三方，不会用于广告投放，不会进行跨站追踪。

浏览权限说明

扩展请求 HTTPS 网页资源访问权限，是为了在用户主动通过右键菜单选择图片后读取该图片资源。扩展不会自动扫描网页内容，不会收集完整浏览历史。

远程代码说明

扩展代码随扩展包一起发布，不从远程服务器加载或执行 JavaScript 代码。扩展使用网络请求仅用于调用用户配置的 AI API 服务和读取用户主动选择的图片资源。

儿童与敏感内容

扩展面向普通生产力和创作辅助场景，不以儿童为目标用户。用户应确保其上传、分析和生成的内容符合法律法规、Chrome Web Store 政策以及所使用 AI 服务提供商的服务条款。

用户控制

用户可以随时打开扩展设置页修改或清空 API Key。用户也可以通过 Chrome 扩展管理页面清除本扩展的本地数据，或直接卸载扩展。

政策更新

如果扩展的数据处理方式发生变化，本隐私政策会同步更新，并在 Chrome Web Store 上架资料中使用最新版本的公开链接。
